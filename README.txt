INVENTORY MANAGEMENT SYSTEM

TECHNOLOGIES USED:
1. Java
2. HTML
3. CSS
4. JavaScript

-----------------------------------------

HOW TO RUN BACKEND

1. Open terminal inside backend/src
2. Compile all files

javac *.java

3. Run program

java Main

-----------------------------------------

HOW TO RUN FRONTEND

1. Open frontend folder
2. Open index.html in browser

-----------------------------------------

1. Product Management
2. Supplier Management
3. Stock Tracking
4. Order Placement
5. Automatic Stock Reduction
6. Refill Orders from Suppliers
7. Sidebar Navigation
8. Responsive UI
